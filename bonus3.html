Level 1

1. 스마트 에어컨 제어 시스템

"실내 온도가 너무 높으면 에어컨을 켜고, 적당하면 끄는 자동화 로직을 만듭니다."
  • currentTemp 변수를 선언하고 숫자 30을 할당하세요.
  • 온도가 28도 이상이면 "에어컨을 가동합니다."를 출력합니다.
  • 그렇지 않으면 "쾌적한 온도입니다."를 출력하세요.
// 이곳에 코드를 작성해 보세요.
✅ 정답 예시 보기 
let currentTemp = 30;

if (currentTemp >= 28) {
    console.log("에어컨을 가동합니다.");
} else {
    console.log("쾌적한 온도입니다.");
}
Level 2

2. 버스 요금 계산기

"나이에 따라 달라지는 버스 요금 체계를 구현합니다."
  • age 변수에 학생의 나이를 넣습니다.
  • 20세 이상: "성인: 1,500원"
  • 14세 ~ 19세: "청소년: 1,000원"
  • 8세 ~ 13세: "어린이: 500원"
  • 7세 이하: "미취학 아동: 무료"
// else if 문을 사용하여 계단식 조건을 만드세요.
✅ 정답 예시 보기 
let age = 16;

if (age >= 20) {
    console.log("성인: 1,500원");
} else if (age >= 14) {
    console.log("청소년: 1,000원");
} else if (age >= 8) {
    console.log("어린이: 500원");
} else {
    console.log("미취학 아동: 무료");
}
Level 3

3. 특별 할인 쿠폰 발급 로직

"로그인 상태를 먼저 확인하고, 조건에 맞는 우수 회원에게만 혜택을 줍니다."
  • isLoggedIn(true/false), purchaseAmount(숫자), isVip(true/false) 변수를 각각 선언하세요.
  • 1단계: 로그인이 안 되어 있으면 "로그인이 필요합니다."를 출력합니다.
  • 2단계: 로그인된 경우, 구매 금액이 50,000원 이상이거나 VIP라면 "쿠폰 발급!"을 출력합니다.
  • 3단계: 둘 다 해당 안 되면 "대상자가 아닙니다."를 출력합니다.
// 중첩 if문과 논리 연산자(||)를 조합해 보세요.
✅ 정답 예시 보기 
let isLoggedIn = true;
let purchaseAmount = 45000;
let isVip = true;

if (isLoggedIn === true) {
    if (purchaseAmount >= 50000 || isVip === true) {
        console.log("특별 할인 쿠폰 발급!");
    } else {
        console.log("대상자가 아닙니다.");
    }
} else {
    console.log("로그인이 필요합니다.");
}
