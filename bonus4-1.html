🔄 JS 반복문 실무 트레이닝

단순 반복 업무는 컴퓨터에게! 백지 상태에서 자동화 로직을 설계해 보세요.

Level 1

1. 배수 콜렉터 (for)

"1부터 30까지의 숫자 중에서 3의 배수만 골라내는 자동화 로직을 작성합니다."
  • for 반복문을 사용하여 1부터 30까지 숫자를 순회하세요.
  • 숫자가 **3의 배수일 때만** 콘솔에 해당 숫자를 출력합니다.
  • 나머지 연산자(%)를 활용해 조건문을 구성해 보세요.
// for 문과 if 문을 조합하여 작성해 보세요.
✅ 정답 예시 보기 
for (let i = 1; i <= 30; i++) {
    if (i % 3 === 0) {
        console.log(i);
    }
}
Level 2

2. 홀수 필터링 합계 (continue)

"1부터 15까지의 숫자 중 홀수만 더하되, 특정 숫자(7)는 제외하고 합산합니다."
  • 합계를 저장할 total 변수를 0으로 초기화하여 선언하세요.
  • 숫자가 **7인 경우**에는 continue를 사용하여 합산 과정을 건너뛰세요.
  • 7을 제외한 나머지 **홀수**들만 total에 누적해서 더하세요.
  • 마지막에 total의 최종 결과값을 출력하세요.
// continue 키워드를 사용하여 흐름을 제어해 보세요.
✅ 정답 예시 보기 
let total = 0;

for (let i = 1; i <= 15; i++) {
    if (i === 7) {
        continue; // 7은 건너뜀
    }
    if (i % 2 !== 0) {
        total += i; // 홀수만 더함
    }
}

console.log("최종 합계: " + total);
Level 3

3. 긴급 정지 카운트다운 (while)

"10부터 역순으로 숫자를 출력하다가, 시스템 결함(숫자 3)이 발견되면 즉시 중단합니다."
  • while 문을 사용하여 10부터 1까지 숫자를 줄여가며 출력하세요.
  • 숫자가 **3에 도달하면** "긴급 정지! 루프를 탈출합니다."를 출력하고 break하세요.
  • 반복문이 안전하게 종료될 수 있도록 변수의 값을 감소시키는 식을 잊지 마세요.
// while 문과 break 키워드를 사용하여 작성해 보세요.
✅ 정답 예시 보기 
let count = 10;

while (count >= 1) {
    if (count === 3) {
        console.log("긴급 정지! 루프를 탈출합니다.");
        break; 
    }
    console.log(count);
    count--; // 값 감소 필수
}
